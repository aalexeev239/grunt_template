Что делать, чтобы начать работать с Grunt?

  1. Запустите терминал и перейдите в нужную директорию cd projectname
  2. Убедитесь в наличии стартового файла package.json
  3. Запустите команду npm install
  4. Установите командную строку npm install -g grunt-cli
  5. Очередь Gruntfile.js появиться на сцене
  6. Установи плагины:

    • load-grunt-tasks

      npm install --save-dev load-grunt-tasks

      Не паримся с подгрузкой заданий

      Прописываем require('load-grunt-tasks')(grunt); вместо grunt.loadNpmTasks('grunt-task') в Gruntfile.js

    • npm install grunt-contrib-concat --save-dev

      Concat

      npm install grunt-contrib-concat --save-dev

      собирает все файлы в один

      Пропиши 'concat' в задания

    • Uglify

      npm install grunt-contrib-uglify --save-dev

      Сжать js

      Пропиши 'uglify' в задания

    • JShint

      npm install grunt-contrib-jshint --save-dev

      Заставляет тебя плакать, глядя на код

      Пропиши grunt jshint в терминале

    • Imagemin

      npm install grunt-contrib-imagemin --save-dev

      Ужимает картинки

      Вызов 'imagemin' в терминале

    • Watch

      npm install grunt-contrib-watch --save-dev

      Описание

      Пропиши 'код' в задания

    • Less

      npm install grunt-contrib-less --save-dev

      Описание

      Пропиши 'код' в задания

    • Autoprefixer

      npm install grunt-autoprefixer --save-dev

      Префиксы на css

      Иногда обновляй базу префиксов npm update caniuse-db

    • CSS-min

      npm install grunt-contrib-cssmin --save-dev

      Минификация Css

      Пропиши 'код' в задания

      plugin

      Описание

      Пропиши 'код' в задания
